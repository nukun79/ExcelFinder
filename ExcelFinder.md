# ExcelFinder 버튼 기능 한줄 요약

- `Excel Folder 찾기...` 버튼: 엑셀 루트 폴더를 선택하며 마지막 탐색 위치를 기억해 다시 엽니다.
- `JSON Folder 찾기...` 버튼: JSON 폴더를 선택하며 마지막 탐색 위치를 기억해 다시 엽니다.
- `새로고침` 버튼: JSON 목록을 다시 읽고 현재 검색어 기준 검색 결과까지 함께 재조회합니다.
- `검색` 버튼: 검색어로 엑셀 전체를 스캔해 파일/시트/행번호/행내용 결과를 갱신합니다.
- `검색어` 입력: UID 또는 일반 검색어를 입력해 검색 기준으로 사용합니다.
- `모든 내용 검색` 체크박스: ON이면 모든 셀 포함검색, OFF이면 `UID` 헤더 컬럼만 정확 일치 검색합니다.
- `JSON 목록 검색` 입력: JSON 파일 목록을 실시간 필터링합니다.
- `검색결과 검색` 입력: 검색 결과를 파일명/시트/행번호/행내용으로 실시간 필터링합니다.
- `JSON 목록 더블클릭`: 선택한 JSON 파일을 기본 연결 프로그램으로 엽니다.
- `검색결과 더블클릭`: 선택한 Excel 파일을 기본 연결 프로그램으로 엽니다.
- `검색결과 CO 표시`: 체크아웃 상태를 원형 색상으로 표시(연두=체크아웃, 빨강=미체크아웃)합니다.
- `검색결과 우클릭 > 에디터 열기`: 선택 행 기준으로 Excel 에디터 창을 엽니다.
- `검색결과 우클릭 > 히스토리`: Perforce 히스토리 창을 열어 리비전 목록을 조회합니다.
- `검색결과 우클릭 > 파일 위치 열기`: 탐색기에서 해당 파일 위치를 열고 파일을 선택합니다.
- `검색결과 우클릭 > Perforce 체크아웃`: `p4 edit`를 실행하고 성공 시 CO 상태를 즉시 갱신합니다.
- `검색결과 우클릭 > Who?`: 체크아웃 상태일 때만 표시되며 해당 파일을 연 Client Workspace명을 팝업으로 보여줍니다.
- `설정 > Perforce 정보 설정...`: Client/Host/Root/Stream을 입력·적용하고 `p4 info` 전체 정보를 하단에 표시합니다.
- `머지` 메뉴 버튼: 엑셀 머지 창을 열어 기준/비교 파일을 선택하고 차이를 관리합니다.
- `머지 창 > 차이점 보기` 버튼: 두 파일의 값이 다른 셀만 목록으로 추출합니다.
- `머지 창 > 머지` 버튼: 결과값 컬럼 기준으로 `_merged` 파일을 생성하고 셀 타입/속성을 유지합니다.
- `머지 창 차이 목록 우클릭 > 기준 에디터`: 기준 파일 에디터를 선택 셀 위치로 바로 엽니다.
- `머지 창 차이 목록 우클릭 > 비교 에디터`: 비교 파일 에디터를 선택 셀 위치로 바로 엽니다.
- `에디터 > 저장` 버튼: 수정한 셀 값을 엑셀 파일에 저장합니다.
- `에디터 > Diff` 버튼: 원본 대비 변경 셀(행/열/이전값/현재값)을 확인합니다.
- `에디터 > 체크아웃` 버튼: 현재 파일을 Perforce 체크아웃합니다.
- `에디터 > 체크인` 버튼: 확인 팝업에서 Description(접두사 포함)과 변경점을 검토/수정 후 제출합니다.
- `에디터 > 리버트` 버튼: 확인 팝업에서 변경점을 확인 후 되돌립니다.
- `체크인 접두사` 입력: 체크인 Description 앞 고정 접두사를 저장해 재사용합니다.
- `체크아웃 상태 표시`: 에디터 상단에서 현재 파일의 체크아웃 ON/OFF 상태를 표시합니다.
- `히스토리 창 우클릭 > Diff` 버튼: 선택 리비전과 이전 리비전의 차이를 별도 창에서 보여줍니다.
- `히스토리 창 우클릭 > Download` 버튼: 선택 리비전 파일을 Save As로 다운로드합니다.
- `버전 표시(v숫자)`: 하단에 현재 빌드 버전을 표시하며 빌드 시 자동 증가 버전 규칙을 따릅니다.
- `창/레이아웃 저장`: 메인 창 크기와 JSON 목록 높이를 저장하고 다음 실행 시 복원합니다.
- 히스토리 Diff 창 표시를 셀 주소 대신 라인 번호와 컬럼명(헤더 우선)으로 변경.
- 히스토리 Diff 창에 찾기 입력/버튼을 추가하고 클릭(또는 Enter) 시 검색 일치 항목만 표시하도록 개선.
- 히스토리 Diff 창 우클릭 에디터 열기 기능을 추가하고 선택 라인으로 이동 및 해당 라인의 변경 셀 강조 표시를 지원.
- 히스토리 Diff 결과 항목 더블클릭 시 선택 리비전 Excel 파일을 기본 프로그램으로 열도록 추가.
- 히스토리 Diff 창에 UID 컬럼을 추가하고 라인별 UID 값을 함께 표시하도록 개선.
- 검색결과에서 에디터 열기 시 대상 라인 강조색을 더 진하게 하고 글자 강조를 추가해 가시성을 강화.
- 메인 상단에 Json추출 메뉴를 추가하고 Data_ 파일명 규칙 기반 체크목록/All 토글/선택 항목 그룹 Json 추출 기능을 구현.
- Json추출 시 헤더가 #로 시작하는 컬럼은 제외하고 추출 후 창이 닫히지 않도록 동작을 변경.
- Json추출 결과 포맷을 행 객체 배열([ {...} ]) 형태로 변경하고 그룹 파일은 데이터 행만 병합 출력하도록 수정.
- Json추출 창에서 Excel/Json 경로를 직접 수정 가능하게 하고 각 경로 옆 찾기 버튼을 추가.
- Json추출 직렬화 옵션을 조정해 한글이 \u 이스케이프가 아닌 원문 UTF-8로 저장되도록 개선.
- Json추출 시 체크박스 편집 상태를 먼저 커밋해 체크된 이름 항목만 정확히 추출되도록 보강.
- Json추출 시 각 Excel 파일에서 첫 번째 시트만 읽어 데이터 추출하도록 변경.
- 메인 메뉴에 JsonDiff를 추가하고 Source/Target JSON 비교 후 차이점만 목록으로 표시하는 창 기능을 구현.
- Json추출 메뉴/창 명칭을 JsonExporter로 변경하고 완료 시 생성된 JSON 파일명 목록 확인 팝업을 추가.
- JsonExporter의 Excel/Json 경로를 설정에 저장/복원해 창 재오픈 시 마지막 사용 경로를 유지하도록 개선.
- 메인 상단 메뉴의 머지 항목명을 ExcelMerge로 변경.
- JsonExporter에서 체크된 이름 목록도 설정에 저장해 재실행 시 체크 상태를 자동 복원하도록 개선.
- JsonDiff 창에 Source/Target Path 기반 폴더 비교를 추가해 같은 JSON 파일끼리 일괄 비교하고 차이점만 목록에 누적 표시하도록 확장.
- JsonDiff 창 레이아웃을 조정해 비교 버튼 영역이 Path 입력 섹션 바로 아래에 붙어 보이도록 개선.
